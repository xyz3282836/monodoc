## 电视领域

随着互联网的发展，家庭电视也逐渐在升级变化，目前电视的视频娱乐主要来源三大阵营：DVB、IPTV、OTT。

|      | 业务运营方    | 网络传输           | 核心终端        | 核心优势              |
| ---- | ------------- | ------------------ | --------------- | --------------------- |
| DVB  | 广电          | HFC[专网]          | 广电机顶盒      | 信号传输/下沉覆盖     |
| IPTV | 电信运营商    | 运营商城域网[专网] | 运营商机顶盒    | 可直播可点播/支持回看 |
| OTT  | OTT集成牌照方 | 运营商[公网]       | 互联网电视/盒子 | 丰富的互联网公网服务  |

OTT是“Over The Top”的缩写，用在电视行业上是指通过公共网络向用户提供内容分发业务。通过广电的有线网开展OTT服务不需要建专网，同时OTT内容的生产制作成本降低，这为传输内容的极大丰富提供了可能，而丰富的内容正是吸引观众的基础。
