## 基础

\d 数字
\w 数字或者字母
\s 空格
\. 任意字符 \* 任意个，包含 0
\+ 至少 1 个
? 0 或者 1 个
{n} n 个
{n,m} n-m 个

[] 表示范围

^ 开头

$ 结束

正则匹配默认是贪婪匹配，也就是匹配尽可能多的字符

```bash
re.match(r'^(\d+)(0*)$', '102300').groups()
'102300',''
```

\d+采用贪婪匹配，需要加?来采用非贪婪匹配

```bash
re.match(r'^(\d+?)(0*)$', '102300').groups()
'1023'，'00'
```
